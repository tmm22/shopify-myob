Privacy Policy - Shopify API to MYOB

Last updated: January 16, 2026

Data Collection

This Chrome extension collects and processes customer data to facilitate transfer between Shopify and MYOB systems. We are committed to protecting your privacy and being transparent about data handling practices.

Types of Data Collected:

  • Customer Information: Name, email address, and phone numbers
  • Billing Address: Street address, city, state/province, postal code, and country
  • Shipping Address: Street address, city, state/province, postal code, and country
  • Order Data: Shopify order information used to populate customer fields

Data Storage

All data is stored locally in your browser using Chrome's storage API. This means:

  • Data never leaves your browser or computer
  • No data is transmitted to external servers
  • Data is only accessible to this extension on your browser
  • Data persists until manually cleared by the user

Data Usage

Collected data is used exclusively for:

  • Auto-filling MYOB customer creation forms with Shopify order data
  • Converting Australian state names to standard abbreviations
  • Maintaining temporary storage for data transfer between platforms

Data Sharing

We do not share, sell, or distribute your data to any third parties.

The extension only operates within your browser and does not transmit data to external services, including:

  • No data analytics services
  • No advertising networks
  • No third-party data processors
  • No external APIs except Shopify and MYOB (which you are already logged into)

Data Security

We implement security measures to protect your data:

  • All data is stored locally using Chrome's secure storage API
  • Data is encrypted at rest using Chrome's built-in encryption
  • No sensitive data is logged or transmitted
  • Extension follows Chrome Web Store security best practices

Data Retention

Customer data is retained until:

  • Manually cleared by the user via the extension popup
  • Browser data is cleared by the user
  • Extension is uninstalled (which removes all stored data)

User Rights and Controls

You have complete control over your data:

  • View: See all stored customer data in the extension popup
  • Delete: Clear all stored data with one click
  • Export: Data can be copied manually before deletion
  • Disable: Extension can be disabled or removed at any time

Third-Party Services

This extension interacts with services you are already logged into:

  • Shopify: Accesses order data using your existing authentication
  • MYOB: Populates forms using your existing session

The extension does not store or transmit any credentials for these services.

Debug Mode

The extension includes an optional debug mode that:

  • Logs technical details to the browser console
  • Is disabled by default and must be manually enabled
  • Does not transmit any debug information externally
  • Logs are stored only in browser memory and cleared on browser close

Changes to This Policy

We may update this privacy policy occasionally. Changes will be:

  • Posted here with an updated "last modified" date
  • Reflected in extension updates with user notification
  • Effective immediately upon posting
